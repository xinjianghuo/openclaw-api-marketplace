# Gumroad License Keys 设置定位

根据常见Gumroad界面，License Keys 通常在这些位置:

## 位置1: Product Settings 区域 (最可能)

在创建/编辑页面，向下滚动到 **Product Settings** 区域（通常在描述框下方）：

```
┌─────────────────────────────────────┐
│ Description                          │
├─────────────────────────────────────┤
│ Product Settings                     │
│  • Language: [English]               │
│  • License keys [OFF/ON]             │  ← 找这个！
│  • Preview: [ON]                     │
│  • max $: [ ]                        │
│  • Category: Software                │
│  • Tags: [ ]                         │
└─────────────────────────────────────┘
```

**操作**:
- 找到 `License keys` 开关，点击打开 (变成蓝色/ON)
- 或点击 `License keys` 文字本身，展开设置

---

## 位置2: Advanced / More options

如果在主区域没有，找:
- 一个按钮或链接: `Show all settings`、`Advanced`、`More options`
- 点击后展开更多选项，License Keys 通常在底部

---

## 位置3: Delivery / Content 标签

左侧或顶部可能有标签栏:
- `Content` / `Delivery` / `Settings`
- 点击这些标签切换到对应设置页面
- License Keys 通常在 **Settings** 标签下

---

## 设置选项 (打开后)

License Keys 有4个配置项:

1. **Format** (格式)
   - 选: `Random characters`

2. **Length** (长度)
   - 输入: `16`

3. **Delivery** (交付方式)
   - 选: `Automatically deliver the license key to the customer`

4. **Custom license key template** (可选)
   - 留空即可

---

## 🔎 找不到怎么办？

**请用浏览器搜索 (Ctrl+F / Cmd+F)**:
- 按 `Ctrl+F`，搜索关键词: `license`
- 会高亮所有包含 "license" 的文本
- 找到的那个就是设置位置

**或告诉我页面截图/描述**:
- 截图整个页面 (包含左侧菜单)
- 或描述: "我看到 X、Y、Z 选项，但没有 license"

我将精确定位。
