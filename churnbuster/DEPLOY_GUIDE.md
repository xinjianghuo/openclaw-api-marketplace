## 📦 第一步：打包可执行文件 (你来做)

### 1. 打开命令提示符 (cmd)
```cmd
cd D:\OpenClaw\.openclaw\workspace\churnbuster
```

### 2. 安装依赖
```cmd
npm install
```
等待完成 (~2分钟)

### 3. 运行测试 (确保一切正常)
```cmd
node test.js
```
预期输出: `✅ 10/10 tests passed`

### 4. 打包成 .exe
```cmd
node build.js
```
输出文件: `dist/churnbuster.exe` (约 20-30 MB)

---

## 🌐 第二步：部署销售页面 (3种方式)

### 方式A: GitHub Pages (免费, 推荐)
1. 创建一个新 GitHub 仓库: `churnbuster-sales`
2. 上传 `SALES_PAGE_CN.html`
3.  Settings → Pages → 选择 `/root` 分支 → 保存
4. 访问: `https://yourusername.github.io/churnbuster-sales/`

### 方式B: 直接本地打开
双击 `SALES_PAGE_CN.html` 在浏览器中查看 → 截图或发送给朋友

### 方式C: Vercel / Netlify (免费托管)
拖拽 `SALES_PAGE_CN.html` 到 [vercel.com](https://vercel.com) 上传 → 自动生成URL
