Frequently Asked Questions

Is this tool safe? Will it leak my API keys? +

Completely safe. ChurnBuster runs 100% locally. No data is uploaded to any server. All scanning and report generation happen on your machine. The code is simple Node.js that you can inspect yourself.

How accurate is the detection? Will it give false positives? +

We provide a confidence score (0-100%). For exact matches (full API keys, specific URLs), accuracy is >90%. Low-confidence items are for reference only—verify manually before acting.

What if it doesn't detect a subscription I know I have? +

ChurnBuster only scans local files and environment variables. Subscriptions paid via mobile app stores, corporate accounts, or bank direct debits won't leave local traces. Combine with bank statement reviews for best results.

Can it automatically cancel subscriptions for me? +

No. Auto-cancellation is too risky—could cause unintended consequences or disputes. ChurnBuster gives you guides so you retain full control and cancel safely.

Does it support macOS or Linux? +

v1.0 is Windows x64 only. If you're on macOS/Linux, you can compile from source (requires Node.js). If there's enough demand, we'll release cross-platform versions in v2.0.

What's the refund policy? +

Full refund within 30 days, no questions asked. If you're unhappy with the software for any reason, simply email a refund request.
