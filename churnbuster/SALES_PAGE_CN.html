常见问题

这个工具安全吗？会泄露我的API密钥吗？ +

完全安全。ChurnBuster 在本地运行，不上传任何数据到服务器。所有的扫描和报告生成都在你的电脑上完成。代码是开源的，你可以自行审查。

检测准确吗？会不会误报？ +

我们为每个服务设定了置信度（0-100%）。对于明确匹配（如完整的API密钥、特定URL），准确率 >90%。低匹配项仅供参考，建议手动核实后再操作。

如果没检测到我确实有的订阅怎么办？ +

ChurnBuster只能扫描本地文件和环境变量。通过手机App Store订阅、公司统一采购、银行代扣的订阅不会留下本地痕迹。建议结合银行账单定期检查。

它能自动帮我取消订阅吗？ +

不会。自动取消风险太高，可能导致误删或纠纷。ChurnBuster提供详细的手动取消指南，你保留最终控制权，安全可靠。

支持 macOS 或 Linux 吗？ +

v1.0 仅支持 Windows x64。如果你有 macOS 或 Linux 环境，可以自行从源码编译（提供Node.js环境）。如果需求大，我们会在 v2.0 推出跨平台版本。
