Frequently Asked Questions

Is this tool safe? Does it steal my data? +

ChurnBuster runs 100% locally. It never sends your data to any server. All scanning and report generation happens on your machine. The code is simple Node.js that you can inspect yourself.

How accurate is the detection? +

We provide a confidence score (0-100%). Known patterns like API keys or login URLs score 90%+. Newer or ambiguous patterns may score lower. Always verify manually before taking action.

What if I don't see a subscription I know I have? +

The tool scans only local files and environment variables. Some subscriptions (like those paid via mobile app stores or corporate accounts) may not leave local traces. Check your bank statements for all recurring charges.

Can it automatically cancel subscriptions? +

No. Automatic cancellation is risky and could lead to unintended consequences. ChurnBuster gives you guides — you remain in control and can cancel safely by following the steps.

What about macOS / Linux? +

v1.0 is Windows-only (most common desktop OS). macOS and Linux versions may be released in v2.0 if there's demand. You can still run the Node.js source on those platforms if you have Node.js installed.
