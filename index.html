🔄 零成本验证流程（5天）

Day 1-2：问题验证（Mom Test）

目标：找到 15 个愿意付费的真实用户

  1. 在 Reddit (r/microsaas, r/indiehackers) 或 HN 搜索痛点
  2. 私信作者，问 5 个关键问题：
    • 每周花多少时间在这个问题上？
    • 现有解决方案是什么？
    • 痛苦程度 1-10？
    • 愿意付多少钱/月？
    • 如果现有解决方案，会立即试用吗？
  3. 记录回复，筛选有效访谈
门槛：≥15 个有效回复，平均痛苦 ≥7，平均支付意愿 ≥$20
通过后 → 进入 Day 3

Day 3：Landing Page（零成本）

工具：GitHub Pages + Jekyll（完全免费）

页面要素：

  • 标题：[具体问题] 的自动化解决方案
  • 副标题：节省 X 小时/周，价格 $Y/月
  • CTA：输入邮箱获取早期访问（50%折扣）
  • 用 Google Forms 或 Typeform 收集邮箱

Day 4：分发测试

渠道（选择 2-3 个）：

  • 相关 subreddit（遵守 self-promotion 规则）
  • Indie Hackers 论坛
  • Product Hunt "Upcoming"
  • Twitter/LinkedIn 定向 outreach

目标：24h 内 ≥30 个邮箱注册
转化率 >10% → 进入 MVP 构建

Day 5：决策

if ≥15 邮箱 & 平均支付意愿 ≥$20
then 用 OpenClaw coding-agent + gh-issues 构建 MVP（2周冲刺）

else pivot 到下一个机会

🛠️ ClawHub 技能发布清单

1. 技能结构

skills/<skill-name>/
├── SKILL.md          # 技能说明（必须）
├── README.md         # 用户快速入门（必须）
├── HOOK.md           # Hook 清单（必须）
├── scripts/
│   └── main.js       # 入口脚本（必须）
└── references/       # 可选（文档、指南）

2. 定价策略

  • 基准价：$29/月
  • 试用期：7 天免费
  • 年付折扣：$299/年（省 1 个月）
  • 终身：$299（仅限前 50 用户）

3. 打包

cd skills/<skill-name>
zip -r ../<skill-name>-v1.0.zip *

4. 上传到 ClawHub

  1. 登录 clawhub.ai
  2. Submit Skill → 上传 zip
  3. 填写元数据（名称、描述、分类、标签）
  4. 上传图标（512x512 PNG）和截图
  5. 提交审核（1-3 天）

5. 收益

  • 分成：70-80% 归开发者
  • 支付：月结，PayPal/Stripe
  • 分析：仪表盘显示下载量、收入、评价

❓ 常见问题

Q: 没有 Reddit API 怎么办？

A: 用 RSS 订阅（r/microsaas/.rss）或在 HN Ask 搜索。Mom Test 也可以用 Twitter/LinkedIn 进行。

Q: 技能审核不通过？

A: 确保：HOOK.md 存在且格式正确；scripts/ 有可执行文件；SKILL.md 完整；无外部依赖。

Q: 如何推广？

A: 在相关 subreddits 分享；Indie Hackers 论坛；Product Hunt Launch；Twitter/LinkedIn 内容营销；OpenClaw Discord 社区。

Q: 需要官方认证吗？

A: 不需要。OpenClaw 技能无需认证即可发布和销售。

Q: 价格如何定？

A: 参考：

  • 简单工具：$9-19/月
  • 中等复杂度：$29-49/月
  • 高级工作流：$79-149/月

建议起点：$29/月 + 7 天试用

开始行动：从 Mom Test 或第一个 ClawHub 技能做起。
