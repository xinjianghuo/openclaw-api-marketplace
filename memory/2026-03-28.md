**阻塞**: GitHub push 需要 PAT (Personal Access Token)  
**解决**: `git-auto-push.ps1` 交互式输入 PAT，自动配置 remote  
**用户动作**: 运行脚本时粘贴 token

**阻塞**: Reddit 发帖需要 API credentials  
**解决**: 创建 `.env` 文件，填入 Reddit App credentials  
**用户动作**: 在 https://www.reddit.com/prefs/apps 创建 app，复制到 `.env`

**阻塞**: 计划任务需要管理员权限  
**解决**: 右键 "Run with PowerShell as Administrator"  
**用户动作**: 以管理员身份运行 `setup-tasks.ps1 -CreateAll`
